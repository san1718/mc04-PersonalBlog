## Overview
**Personal Blog** is a simple and elegant blogging application that allows users to share their thoughts and experiences. The app supports both light and dark mode for a customized reading and writing experience. Users can submit blog posts, view past entries, and delete them as needed. Blog posts are stored in **localStorage**, ensuring persistence across sessions.

## Features
✅ Dark and Light mode toggle  
✅ Submit personal blog posts  
✅ View a list of blog posts  
✅ LocalStorage-based persistence  
✅ Delete/Clear all posts  

## Technologies Used
- **Front-end:** HTML, CSS, JavaScript  
- **Storage:** LocalStorage  

## Installation & Usage
For local use:
### 1️⃣ Clone the Repository  
```sh
git clone git@github.com:san1718/mc04-PersonalBlog.git
cd mc04-PersonalBlog
```

### 2️⃣ Open the Application  
Simply open `index.html` in your preferred browser.  

OR, start a local server:  
```sh
npx live-server
```

## Summary
**Personal Blog** provides a minimalist and user-friendly blogging experience, allowing users to write, store, and manage their posts effortlessly. With a built-in dark mode and persistent storage, users can revisit their thoughts at any time.
